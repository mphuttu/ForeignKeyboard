Note: For right-to-left languages (Arabic, Farsi, Hebrew) the output text box automatically switches to RTL text direction so that characters appear in the correct order.

The Shift Button

The Shift button toggles between lowercase and uppercase for languages that support both (Greek and Russian). The button label changes to Shift ON when uppercase mode is active. Clicking it again returns to lowercase. For Arabic, Farsi and Hebrew the Shift button is disabled because these scripts do not distinguish letter case.
