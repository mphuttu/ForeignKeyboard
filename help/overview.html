Overview – What is Foreign Keyboard?

Foreign Keyboard is a Windows desktop application designed for users who occasionally need to type text in a language that uses a non-Latin alphabet – such as Arabic, Farsi, Greek, Hebrew or Russian – without having to reconfigure their operating system keyboard layout.

Main Features

  • On-screen virtual keyboard for five non-Latin scripts.
  • Automatic right-to-left (RTL) text direction for Arabic, Farsi and Hebrew.
  • Shift key support for languages that distinguish upper- and lowercase (Greek, Russian).
  • Extended polytonic (ancient) Greek: breathing marks, accent marks, iota subscript and final sigma for typing Classical and Byzantine Greek texts.
  • Comprehensive set of special character and punctuation buttons.
  • One-click Copy to clipboard.
  • Physical keyboard pass-through: typing on your real keyboard also inserts the mapped foreign characters.
  • Tooltip hints on every button.

System Requirements

  • Windows 10 or Windows 11 (64-bit)
  • No additional fonts need to be installed – all required Unicode characters are included in the default system fonts.
