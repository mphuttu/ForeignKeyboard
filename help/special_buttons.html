Special Buttons Reference

The rows of small buttons above the alphabet keyboard area give you quick access to common punctuation, editing actions and special characters. Hover the mouse over any button to see its tooltip description.

Action Buttons

ButtonAction
CopyCopies all text in the output box to the Windows clipboard.
ClearClears (deletes) all text in the output box.
BackspaceDeletes the character before the cursor (or the selected text).
DeleteDeletes the character after the cursor (or the selected text).
HomeMoves the cursor to the very beginning of the text.
EndMoves the cursor to the very end of the text.
SpaceInserts a space character at the cursor position.
