Using the Physical Keyboard

In addition to clicking the on-screen buttons you can type using your physical keyboard. Foreign Keyboard maps the letter keys A–Z and the digit keys 0–9 to the first letters of the selected alphabet in alphabetical order.

Key Mapping

The mapping is sequential: physical key A produces the first letter of the selected alphabet, B produces the second letter, and so on. Hold Shift on the physical keyboard to type uppercase letters (for Greek and Russian).

Example – Greek:
A → α   B → β   C → γ   D → δ … (Shift+A → Α, Shift+B → Β …)
Example – Arabic:
A → ا   B → ب   C → ت   D → ث …

Editing Keys

The following physical keyboard keys work directly in the output text box:

KeyAction
BackspaceDelete character before cursor
DeleteDelete character after cursor
HomeMove cursor to beginning
EndMove cursor to end
Arrow keysMove cursor left / right
Ctrl+ASelect all text
Ctrl+CCopy selected text
Ctrl+VPaste text from clipboard
Ctrl+XCut selected text
Ctrl+ZUndo last change
Note: The physical keyboard mapping only covers the first 36 characters of each alphabet (26 letters A–Z + digits 0–9). If the alphabet has more characters than 36, use the on-screen buttons for the remaining characters.
