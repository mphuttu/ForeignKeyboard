Alphabet Keyboard Area

The large bordered area in the centre of the window is the Alphabet Keyboard Area. When you select a language, the application fills this area with one button per letter of that alphabet.

How It Works

  • Each button displays a single letter in the currently selected script.
  • Click a button to insert that letter at the current cursor position in the output text box.
  • Buttons are laid out from left to right in alphabetical order, wrapping to the next row when the row is full.
  • For right-to-left scripts (Arabic, Farsi, Hebrew) the insertion order is handled automatically by the RTL output text box.

Uppercase / Lowercase

For Greek and Russian, toggle the Shift button above the keyboard area to switch all letter buttons between their lowercase and uppercase forms.

Tip: After inserting characters from the alphabet keyboard you can continue editing the text using the physical keyboard – arrow keys, Backspace, Delete and Home/End all work normally inside the output text box.
