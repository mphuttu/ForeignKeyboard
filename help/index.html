Foreign Keyboard – Help

Welcome to Foreign Keyboard!
Foreign Keyboard is a virtual on-screen keyboard that lets you type characters from Arabic, Farsi (Persian), Greek, Hebrew and Russian using a standard keyboard or mouse clicks.

Quick Start

  1. Select a language from the drop-down list at the top left.
  2. The alphabet keyboard appears in the large area below.
  3. Click any letter button to insert it into the output text box.
  4. Use the special buttons (Copy, Clear, Space, punctuation …) as needed.
  5. Click Copy to copy the finished text to the clipboard.
